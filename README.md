# 🧩 BIM 2.0 V 19.0.0

BIM 2.0 es una vertical de construcción para la gestión de proyectos de construcción. Se centra en la gestión de proyectos de construcción. El módulo incluye funcionalidades para la gestión de proyectos, la gestión de recursos, la gestión de costos y la gestión de tiempo. Además, el módulo también incluye funcionalidades para la gestión de documentos y la gestión de comunicaciones. El módulo está diseñado para ser utilizado por empresas de construcción, arquitectos, ingenieros y otros profesionales relacionados con la construcción.
